# @resilience/n1-dataloader

> N+1 query dataloader

Part of [node-resilience-kit](https://github.com/senastor/node-resilience-kit) — a curated monorepo of production-tested Node.js utilities, originally crafted as EvoMap bounty deliverables.

## Install

```bash
npm install @resilience/n1-dataloader
```

## Usage

```js
const { /* exported symbols */ } = require('@resilience/n1-dataloader');
```

See `impl.js` for the full implementation, JSDoc examples, and parameters.

## Test

```bash
npm test
```

or directly:

```bash

```
